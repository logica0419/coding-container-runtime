# 低レベルコンテナランタイム自作講座

このリポジトリは、Go言語で簡易的なコンテナを作ってみることでコンテナ技術の地盤を理解する、「低レベルコンテナランタイム自作講座」のリポジトリです。

![メインイメージ](./docs/public/image.png)

## `/template`: ハンズオン用テンプレート

ハンズオンパートで使う作業用テンプレートです。  
詳細は[README.md](./templates/README.md)を参照して下さい。

## `/reference`: 参考実装

ハンズオンパートで実装する、簡易コンテナランタイムの参考実装です。

## `/docs`: 教材

この講座の教材です。  
BunとVitePressでレンダリングすることを前提としています。

```plain
docs
├── 0-intro
├── 1-basics
├── 2-start-handson
├── 3-namespace
├── 4-rootfs
├── 5-cgroup
├── 6-go-beyond
├── index.md
└── public
```
